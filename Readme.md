# Starlight Rewind

A little rewind of what happened until now

## Installation
```
$ npm install
```

## Generating rewind
```
$ npm run rewind
```

## Run server
```
$ npm run start
```

## Environment

| Name | Description | Default value |
| ---- | ----------- | ------------- |
| STARLIGHT_REWIND_PORT | The port on which the server will listen | 3000 |
| STARLIGHT_REWIND_PREFIX | Url prefix to use | "" |
